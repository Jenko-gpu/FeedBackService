# Feedback Service (Pet Project)

Проект представляет собой бэкенд-систему для сбора, хранения, модерации и обработки пользовательских отзывов, обращений в поддержку и баг-репортов. Реализована поддержка загрузки изображений и видео.


---

## Основной функционал

- **Работа с отзывами** – создание, просмотр, редактирование, удаление, фильтрация по рейтингу и статусу.
- **Обращения в поддержку** – создание тикетов с категориями и приоритетами, назначение ответственных, смена статусов.
- **Баг-репорты** – регистрация ошибок с указанием severity, компонента, шагов воспроизведения.
- **Модерация** – администраторы могут одобрять или отклонять отзывы с комментариями.
- **Загрузка файлов** – отдельный сервис для загрузки, скачивания и управления метаданными файлов (MinIO/S3).
- **Асинхронные события** – через Kafka передаются уведомления о новых отзывах, изменениях статусов и т.п.

## API (кратко)

Основные эндпоинты (все требуют JWT, кроме публичных):

- `POST /api/v1/feedback` – создать отзыв  
- `GET /api/v1/feedback` – список с пагинацией и фильтрацией  
- `PUT /api/v1/feedback/{id}` – обновить  
- `DELETE /api/v1/feedback/{id}` – удалить  
- `POST /api/v1/support` – создать обращение  
- `PUT /api/v1/support/{id}/status` – изменить статус  
- `POST /api/v1/bug-reports` – создать баг-репорт  
- `GET /api/v1/admin/moderation/pending` – отзывы на модерации  
- `PUT /api/v1/admin/moderation/{id}/approve` – одобрить  

Файловый сервис (отдельный порт):

- `POST /api/v1/files/upload` – загрузить файл  
- `GET /api/v1/files/{id}` – скачать  
- `DELETE /api/v1/files/{id}` – удалить
